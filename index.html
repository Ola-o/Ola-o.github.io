Git Workflow

    What is a commit?

    This command is used to commit all staged files to version history.
    git commit -m"I am trying to commit my code to a staging area"

    What is a branch?

    Git branches are effectively a pointer to a snapshot of your changes.
    git branch

    What is a pull request?

    A pull request is an event in Git where a contributor asks a maintainer of a Git repository to review code they want to merge into a project.
    git pull

    Unix Shell

    How to list the files in a directory.

    The command 'ls' lists files in a given directory.

    How to show and change the working directory.

    The command 'pwd' shows working directory and 'cd' changes working directory.

    How to create a new directory.

    The command 'mkdir' creates a new directory.

    How to create and remove files.

    The command 'touch' 'filename.txt' creates a new file named filename.
    while 'rm' 'filename.txt' removes the file filename.txt.
